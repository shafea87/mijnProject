Front-End

Front-End developers are the ones who take the design and develop it for the webpage that works. The most important skills of the Front-End expert are HTML, CSS and JavaScript language management. HTML and CSS are the basic skills of a web developer whose management is essential. JavaScript is important because it can create dynamic functionalities for websites and as a user can react with sites.

HTML and CSS
HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are the two key technologies for website building by a front-end developer. HTML provides a layout of a page, a CSS (visual and aural) layout for multiple devices. In addition to graphics and scripts, HTML and CSS form the basis for web sites and web applications [1]. HTML Example Figure 1. HTML Example HTML is the language for describing the structure of Web pages (Figure 1). HTML gives authors the means to:
  • Publish online documents with headings, text, tables, lists, photos, etc.
  • Retrieve online information via hypertext links, at the click of a button.
  • Design forms for conducting transactions with remote services, for use in searching for information, making reservations, ordering products, etc.
  • Include spread-sheets, video clips, sound clips, and other applications directly in their documents.

With HTML, authors describe the structure of pages using markup. The elements of the language label pieces of content such as “paragraph,” “list,” “table,” and so on [1].
CSS (Cascading Style Sheets) is the language for describing the presentation of Web pages, including colors, layout, and fonts (Figure 2). It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers [1].
CSS is independent of HTML and can be used with any XML-based markup language. The separation of HTML from CSS makes it easier to maintain sites, share style sheets across pages, and tailor pages to different environments. This is referred to as the separation of structure (or: content) from presentation [1].
CSS helps Web developers create a uniform look across several pages of a Web site. Instead of defining the style of each table and each block of text within a page's HTML, commonly used styles need to be defined only once in a CSS document. Once the style is defined in cascading style sheet, it can be used by any page that references the CSS file. Plus, CSS makes it easy to change styles across several pages at once. For example, a Web developer may want to increase the default text size from 10pt to 12pt for fifty pages of a Web site. If the pages all reference the same style sheet, the text size only needs to be changed on the style sheet and all the pages will show the larger text [8].

JavaScript is the most famous scripting language on web pages and is also used by many non-browser environments, such as node.js and Apache CouchDB. JavaScript is a light dynamic programming language that supports object-oriented programming style. (Figure 3.) In some cases, a very simple site is good, but in situations where interactive features - audio and video, games, scrolling features, page animations - are required, JS is the tool used to implement them [3].

Back-End

Web application implementation is rarely just a front-end development. Also required is a server-side application that includes databases, background drives, and faster performance. The site's back-end consists of a server, an application, and a database. A back-end developer builds and maintains a technology that enables components that together allow the front side of the site to exist [2]. In order for server, application and database to communicate with each other, the back-end programmers use a server-side languages ​​such as PHP, Ruby, Python, Java, Perl and .Net. It is also useful if the developer knows how Wordpress (Wordpress is written in PHP, so its functions, functions, plug-ins, and themes use the same language) or Mongo DB work. To find, store, and edit data, application developers use tools such as MySQL, Oracle and SQL Server. The Back-end developer must have a strong knowledge of Back-End programming, but also an understanding of front-end technologies (such as CSS, HTML, and JavaScript) to communicate with other team members. This role also needs to manage security issues and hosting environments and database management [5]. The back-end developer's job description is extensive (from writing code to database management, optimization, knowledge of the most popular software frameworks), so it is not prepared directly from school, but it takes years of experience to collect skills [6].

Browser development tools usually consist of an Inspector and a JavaScript console. With the help of the inspector, you can see what the HTML of the page impressions shows, which CSS is associated with each element of the page, and you can also edit the HTML and CSS code and see the changes as such. The JS console can be used to view possible errors that the browser is trying to perform in the JS code [4].
